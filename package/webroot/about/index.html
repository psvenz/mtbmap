Description:
The map is a Mountainbike map based on OpenStreetMap data.

See mtb:scale and mountainbike on openstreetmap wiki.

Please contribute with better data and add your favourite mountainbike trails to openstreetmap! If you haven't contributed to Openstreetmap before start here.

Site and custom style sheet is developed by zvenzzon.

Update interval: ~monthly or worse =)

Software used:

  • Maperitive
  • OSMTileMachine
  • Leaflet
  • osmosis
  • osmconvert
  • osmupdate
  • wget
  • nginx
  • stefanocudini-leaflet-gps
  • shramov-leaflet-plugins
